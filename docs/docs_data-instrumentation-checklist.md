# Data Instrumentation & Analytics Checklist

Purpose:
- Ensure events, metrics, and dashboards required to measure success are defined and implemented.

Checklist:
- [ ] Define success metric(s) for the feature with Product and PM
- [ ] Create event & property schema for required signals
- [ ] Confirm event naming and taxonomy align with analytics standards
- [ ] Implement instrumentation and unit/integration tests
- [ ] Create dashboards or charts for key metrics
- [ ] Schedule initial analysis within 2 sprints of release
- [ ] Document any sampling, retention, or privacy considerations
